# 使用方法
## 测试全部case:
```
python -m unittest cnns_test
```

## 指定测试case:
```
python -m unittest cnns_test.TestAlexNet
```

指定测试case和方法：
```
python -m unittest cnns_test.TestAlexNet.test_1n1c
```

## 测试输出形式：

```
======================================================================
xx net result report
======================================================================
of-1n1c -----------------------------------------------------     pass
of-1n4c ----------------------------------------------------- not pass
of-2n4c -----------------------------------------------------     pass
```

通过 `-v`参数指定详细输出报告：
```
python -m unittest -v cnns_test
```
